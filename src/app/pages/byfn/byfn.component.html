Build Your First Netrwork

Firstly take a look at guide for Prerequisites & Samples etc.

Now, when you did it, go open terminal and navigate to this path:

.../fabric-samples/first-network$


Then, go follow these steps:

bubble_chart $ ./byfn.sh generate $ sudo groupadd docker
$ sudo usermod -aG docker $USER
$ newgrp docker
$ docker run hello-world $ ./byfn.sh up
$ ./byfn.sh down

After all, if you comes to some error with message, while you running UP command, like this one:
"FORBIDDEN --config update for existing channel did not pass initial checks..."
Then follow these steps belowe to turn off all docker containers:

$ docker stop $(docker ps -a -q)
$ docker rm $(docker ps -a -q)
$ docker system prune -a
$ docker volume prune
$ docker network prune

You can find full guide for BYFN on official documentation
