Begginer Guide

Here are, less or more, everything you have to know about Hyperledger Fabric as a begginer, in brief!

Firstly, here you can check if you have installed all prereqisites, as quick as possible:

$ sudo apt-get install cURL $ sudo apt-get install docker
$ sudo apt-get install docker-compose $ sudo apt-get install go-lang $ sudo apt-get install nodejs
$ npm install npm@5.6.0 -g $ sudo apt-get install python2



Now, with these comands bellow you can provide Samples, Binaries and Docker Images you need for BYFN:

$ curl -sSL http://bit.ly/2ysbOFE | bash -s -- 1.4.6 1.4.6 0.4.18

Then, we strongly recommend that you read our glossary!

After that, you may wish to get acqualnted with the Hyperledger Fabric architecture network. Let's check it on link bellow.

Also we have this photo that might help you understand the architecture itself.

Futhermore, we suggest you follow the official tutorial of Hyperledger Fabricto write your first application. But before that, it would be nice to read also official chapter for developing application.


Finally, it's time to choose language for SDK you will use in your future projects

SDK is replacement for all those commands that we run from terminal.

So, we wish you good luck in you future work in this field.
